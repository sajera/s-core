    static handleAsAsync(action, callback)
    lib/utilities/utilities.js, line 113

    Method to provide ability to use for asynchronous actions "async" flow.
    Regardless of the action thread used.
    NOTE Context binding to an action is strongly suggested.
    Without binding action will lose the context, as any other callback function.

    NOTE We asume the last arguments its async callback
    Parameters:
    Name Type Description
    action function asynchronous action
    ...arguments * any type and any count of arguments to deleagete it in action
    callback function async callback to use async way
    See:
    • asinc on NPM
    Example
    const handle = Core.utils.handleAsAsync;
 // async flow
 handle(action.bind(ctx),a,r,g,u,m,e,n,t,s,(error, result) => {
     ... // handle using async cllback
 });

    lib/utilities/utilities.js, line 141

    Method to provide ability to use for asynchronous actions "promise" flow.
    Regardless of the action thread used.
    NOTE Context binding to an action is strongly suggested.
    Without binding action will lose the context, as any other callback function.
    Parameters:
    Name Type Description
    action function asynchronous action
    ...arguments * any type and any count of arguments to deleagete it in action
    Returns:
    Promise - - to use promise way
    See:
    • native Promise
    Example
    const handle = Core.utils.handleAsPromise;
 // promise flow
 let promise = handle(action.bind(ctx),a,r,g,u,m,e,n,t,s);
 promise.then(success => console.log('success', success));
 promise.catch(error => console.log('error', error));
