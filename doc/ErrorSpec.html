    new ErrorSpec(id, spec, error)
    lib/errors/errorspec.js, line 49

    ErrorSpec has simple usage.
    Define your errors specification and extend specification of able error list.

    Implementation of Error Interface.

    Parameters:
    Name Type Description
    id String ID of error within specification
    spec Object all data defined as data of error from specification
    error Object previous error to build full stack of errors
    Implements:
    • ErrorInterface
    Author:
    • Sajera
    Example
    ErrorSpec.define('error 501', {
     level: 10,
     status: 501,
     message: 'something broke'
 });
 ErrorSpec.create(id [, error=null]);
